# `mgmake.hxx`

**Source:** `mgmake.hxx`  
**Status:** Generated single-header distribution; current with the modular headers in this archive.

The root `mgmake.hxx` is the distributable amalgamation of the public umbrella header and all project-local headers reachable from it.

## Generation

`tools/amalgamate.py` starts from `include/mgmake/mgmake.hxx`, recursively expands project-local `#include` directives, and leaves standard-library includes intact. Each expanded file is wrapped in comments such as:

```cpp
// ===== begin include/mgmake/cli/entry.hxx =====
// ...
// ===== end include/mgmake/cli/entry.hxx =====
```

Already-visited project headers are replaced with a `skipped duplicate include` comment. A top-level `MGMAKE_SINGLE_HEADER_HXX` guard encloses the result.

## Usage

A build script can include the file directly:

```cpp
#include "mgmake.hxx"

using my_config = mgmake::config::project<my_project>;
MGMK_ENTRY(my_config);
```

The header also exposes `namespace mgmk = mgmake` through the modular umbrella.

## Maintenance rule

Do not edit this file directly. Make changes under `include/mgmake/`, then run:

```sh
python3 tools/amalgamate.py
```

The generated file in this archive matches the result produced from the included modular headers.

## Scope

Only headers reachable from `include/mgmake/mgmake.hxx` are amalgamated. Experimental headers that are not included by the umbrella—currently including the discovery subsystem and `spec/toolchain.hxx`—are therefore absent from the single-header distribution.
